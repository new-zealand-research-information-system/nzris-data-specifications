# NZRIS Link Values
Relationships between records (both within a category and across categories) are created using a numbered “link”. The following is a list of available link values with associated usage - see the attached document or full [Data Specification and Data Dictionary](https://github.com/new-zealand-research-information-system/nzris-data-specifications/blob/master/Data%20Specification%20and%20Data%20Dictionary.pdf) for the correct order when completing the ingestion template.
